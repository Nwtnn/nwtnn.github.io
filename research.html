> FIELD_NOTES // OBSERVATIONS

NOTE_001 // Default Credential Risk

Across Tier 0 labs, the single most common attack vector was misconfigured or default credentials — Telnet with no password, Redis with no auth, FTP with anonymous access. These aren't theoretical risks; they're real-world attack surfaces found in production every day.

NOTE_002 // SQL Injection Basics

The Appointment machine demonstrated how a single unparameterized SQL query can collapse an entire authentication system. A payload like admin'-- is still surprisingly effective on legacy applications.
